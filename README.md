# First Odin Project — Recipes List

This is my first project from **The Odin Project**, focused on practising the basics of **HTML** by building a simple recipes website.

The goal of this project is to create a small recipe list using only HTML.  
It includes a main page with links to individual recipe pages, allowing navigation between different parts of the website.

Besides the common basic HTML structure, I also tried to add a few extra details, such as:

- Links between pages
- Images for each recipe
- Image links where useful
- Better page structure using headings, paragraphs and lists
- Simple navigation back to the homepage
